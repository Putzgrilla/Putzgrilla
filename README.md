## 🚀 Sobre mim

Estudante e desenvolvedor backend com foco em **Java** e **Spring Boot**. Apaixonado por criar sistemas bem arquitetados, trabalho tanto em projetos universitários (incluindo sistemas distribuídos) quanto em projetos pessoais para expandir meu conhecimento em backend, APIs e bancos de dados.

Gosto de explorar novas tecnologias, praticar lógica de programação e me aprofundar em arquitetura de software e padrões de projeto.
